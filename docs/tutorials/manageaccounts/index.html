Manage Accounts

The ownership of CKB is established through keys, the lock script and addresses. Lumos also provides the functions to manage the keys, addresses and the lock script of a CKB account.

Prerequisites#

The following prerequisites apply for managing accounts by using Lumos:

  • The development environment is set up. For more information, see Set Up the Development Environment.
  • The CKB node is installed and started on DEV chain. For more information, see Install a CKB Node.
  • The Lumos packages (@ckb-lumos/base, @ckb-lumos/indexer, @ckb-lumos/helpers, @ckb-lumos/config-manager, @ckb-lumos/hd) are installed.

Environment#

The following examples are verified on Ubuntu 20.04.2. Steps on the other platforms can be adjusted accordingly.

Examples#

Generate Keys#

The HD wallet manager (@ckb-lumos/hd) supports BIP-39 for generating seeds, mnemonic codes (12 words), extended private and public keys.

The mnemonic.generateMnemonic function can generate 12 words mnemonic.

Example:

hellolumos/src/manageaccounts.ts/generateKey
import { mnemonic, ExtendedPrivateKey } from "@ckb-lumos/hd";
export async function generateKey(){
const m = mnemonic.generateMnemonic();
console.log("The mnemonic is",m);
const seed = mnemonic.mnemonicToSeedSync(m);
const extendedPrivateKey = ExtendedPrivateKey.fromSeed(seed);
console.log(extendedPrivateKey);
const publickey = extendedPrivateKey.toExtendedPublicKey().publicKey;
console.log("The public key is", publickey);
}

Convert a Private Key to a Public Key#

The key module of the @ckb-lumos/hd package supports the secp256k1 standard. The key.privateToPublic function can convert a private key to a public key.

Example:

hellolumos/src/manageaccounts.ts/private2Public
import { key } from "@ckb-lumos/hd";
export async function private2Public (
privatekey: HexString
):Promise<HexString> {
const pubkey = key.privateToPublic(privatekey);
console.log("The public key is",pubkey);
return pubkey;
}

Generate the Args from a Public Key#

The key.publicKeyToBlake160 function uses blake160 to extracts the first 20 bytes of a public key hash as the args.

Example:

hellolumos/src/manageaccounts.ts/public2Args
import { key } from "@ckb-lumos/hd";
export async function public2Args (
publicKey: HexString
):Promise<string> {
const args = key.publicKeyToBlake160(publicKey);
console.log("The args is",args);
return args;
}

Recover the Public Key from a Signature and a Signed Message#

Signatures are based on elliptic curve private/public key pairs. The public key is recoverable from the signature.

The key.recoverFromSignature function can recover the public key from a signature and a signed message.

Example:

hellolumos/src/manageaccounts.ts/signature2PublicKey
import { key } from "@ckb-lumos/hd";
export async function signature2PublicKey (
message: HexString,
signature: HexString
):Promise<HexString> {
const pubkey = key.recoverFromSignature(message,signature);
console.log("The public key is",pubkey);
return pubkey;
}

Generate a Keystore File from an Extended Private Key#

A keystore file is an encrypted version of a private key. The keystore file is generated by using the private key and a password that is used to encrypt the keystore file. The ckb-cli tool or wallets like the CKB wallet, Neuron Wallet, can import keystore files to create new accounts.

The Keystore module of the @ckb-lumos/hd package supports to generate keystore files from extended private keys.

Example:

hellolumos/src/manageaccounts.ts/generateKeystore
import { Keystore } from "@ckb-lumos/hd";
export async function generateKeystore(
password: string,
path: string,
name: string,
overwrite: boolean | undefined
){
const m = mnemonic.generateMnemonic();
console.log("The mnemonic is",m);
const seed = mnemonic.mnemonicToSeedSync(m);
const extendedPrivateKey = ExtendedPrivateKey.fromSeed(seed);
console.log("The extendedPrivateKey is", extendedPrivateKey);
const keystore = Keystore.create(extendedPrivateKey,password);
keystore.save(path,{name, overwrite});
}

Generate an XPub Key File from an Extended Private Key#

An xPub Key (extended public key) can be used to derive a unique address for every transaction.

The XPubStore class of the @ckb-lumos/hd package supports to generate xPub key files from extended private keys.

Example:

hellolumos/src/manageaccounts.ts/generateXpubStore
import { ExtendedPrivateKey, XPubStore } from "@ckb-lumos/hd";
export async function generateXPubStore(
extendedPrivateKey: ExtendedPrivateKey,
path: string,
overwrite: boolean | undefined
){
const accountExtendedPublicKey = extendedPrivateKey.toAccountExtendedPublicKey();
const xpubstore = new XPubStore(accountExtendedPublicKey);
xpubstore.save(path,{overwrite});
}

Generate the Address from a Lock Script#

The generateAddress function of the @ckb-lumos/helpers package can be used to generate the address from a specific lock script.

Example:

hellolumos/src/manageaccounts.ts/generateAddressfromLock
import { generateAddress } from "@ckb-lumos/helpers";
export async function generateAddressfromLock(
lockScript:Script,
config: Config
) {
const address = generateAddress(lockScript, {config});
console.log("The address for the lockscript is", address);
}

Get the Lock Script from an Address#

The parseAddress function of the @ckb-lumos/helpers package can be used to get the lock script from an address.

Example:

hellolumos/src/manageaccounts.ts/generatelockFromAddress
import { parseAddress } from "@ckb-lumos/helpers";
export async function generateLockFromAddress (
address:Address
) {
const lockscript = parseAddress(address);
console.log("The lockscript of the address is", lockscript);
}

Generate the Lock Hash from a Lock Script#

The computeScriptHash function generates the hash value for a specific lock script.

hellolumos/src/manageaccounts.ts/generateLockHash
import { utils } from "@ckb-lumos/base";
const { computeScriptHash } = utils;
export async function generateLockHash(
lock:Script
){
const lockHash = computeScriptHash(lock);
console.log("The lockHash is", lockHash);
}

Generate an Account from a Private Key#

hellolumos/src/manageaccounts.ts/generateAccountFromPrivateKey
import { parseAddress} from "@ckb-lumos/helpers";
import { utils, Address, Hash, Script, HexString } from "@ckb-lumos/base";
const { computeScriptHash } = utils;
import { key } from "@ckb-lumos/hd";
export type Account = {
lockScript: Script;
lockHash: Hash;
address: Address;
pubKey: string;
lockScriptMeta?: any;
}
export const generateAccountFromPrivateKey = (privKey: string): Account => {
const pubKey = key.privateToPublic(privKey);
const args = key.publicKeyToBlake160(pubKey);
const template = CONFIG.SCRIPTS["SECP256K1_BLAKE160"]!
const lockScript = {
code_hash: template.CODE_HASH,
hash_type: template.HASH_TYPE,
args: args
};
const address = generateAddress(lockScript);
const lockHash = computeScriptHash(lockScript);
return {
lockScript,
lockHash,
address,
pubKey,
}
}
